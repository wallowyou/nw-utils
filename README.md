# nw-tools


**English** | [中文](./README.zh-CN.md)

## Introduction

Commonly used javascript tool functions

## Documentation

[Documentation Github](https://wallowyou.github.io/modules.html)


## Install

### npm

```shell
npm i nw-tools -S
```

### pnpm

```shell
pnpm i nw-tools -S
```

### yarn

```shell
yarn add nw-tools
```

## Use

### ESM

```ts
import { is } from 'nw-tools'

is('test', 'Number')
```

### CJS

```js
const nwTools = require('nw-tools')

nwTools.is('test', 'Number')
```
