## Introduction

Commonly used javascript tool functions

## Documentation

[Documentation Github](https://wallowyou.github.io/modules.html)


## Install

### npm

```shell
npm i nw-utils -S
```

### pnpm

```shell
pnpm i nw-utils -S
```

### yarn

```shell
yarn add nw-utils
```

## Use

### ESM

```ts
import { is } from 'nw-utils'

is('test', 'Number')
```

### CJS

```js
const nwUtils = require('nw-utils')

nwUtils.is('test', 'Number')
```
