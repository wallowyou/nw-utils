# nw-utils



**中文** | [英文](./README.zh-CN.md)

## 介绍

常用的js工具函数

## 文档

[github文档地址](https://wallowyou.github.io/modules.html)


## 安装

### npm

```shell
npm i nw-utils -S
```

### pnpm

```shell
pnpm i nw-utils -S
```

### yarn

```shell
yarn add nw-utils
```

## 使用

### ESM

```ts
import { is } from 'nw-utils'

is('test', 'Number')
```

### CJS

```js
const nwUtils = require('nw-utils')

nwUtils.is('test', 'Number')
```
