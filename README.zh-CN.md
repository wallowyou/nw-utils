# nw-tools



**中文** | [英文](./README.zh-CN.md)

## 介绍

常用的js工具函数

## 文档

[github文档地址](https://wallowyou.github.io/modules.html)


## 安装

### npm

```shell
npm i nw-tools -S
```

### pnpm

```shell
pnpm i nw-tools -S
```

### yarn

```shell
yarn add nw-tools
```

## 使用

### ESM

```ts
import { is } from 'nw-tools'

is('test', 'Number')
```

### CJS

```js
const nwTools = require('nw-tools')

nwTools.is('test', 'Number')
```
